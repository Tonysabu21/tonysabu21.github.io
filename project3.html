Ancient Horse 3D Reconstruction

Overview

The goal of this project was to recreate an ancient sculpture using only a reference image. Using depth analysis from shadows and structural estimation, I rebuilt the model digitally and prepared it for 3D printing.

What I Did

Built base mesh model for sculpting workflow
Refined geometry using CAD + STL editing tools
Interpreted depth using lighting and shadow analysis
3D printed prototype for real-world validation
Iterative refinement for structural accuracy
Final smoothing and surface finishing

Skills Used

3D Modeling CAD 3D Printing Design Analysis
